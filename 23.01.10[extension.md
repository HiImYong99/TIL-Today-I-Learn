## **extension 요약**

### extension을 사용하여 기존에 존재하는 타입을 확장 시킬 수 있다. 추가로 다른 프로토콜을 채택할 수 있도록 확장 할 수도 있다.

```swift
extension name: protocolName {
	code
}
```

### 이처럼 extension을 활용하면 기능에 따라 코드를 분리 할 수 있어 좋은 코드를 작성하는데 많은 도움이 된다.
